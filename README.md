# StudySync

StudySync is a student productivity and learning management web application designed to help students stay organized and focused throughout their academic journey.

## Features

- Dashboard with productivity statistics
- Task management system
- Class schedule organizer
- Study planner with progress tracking
- Notes management
- Pomodoro focus timer
- Learning analytics dashboard
- Notification center
- Student profile page
- Dark mode support

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js
